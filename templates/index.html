What is DDLR Face Recognition?

DDLR Face Recognition is a web application to detect our team's face from a given image. DDLR itself stands for Devan, Dayu, Leo, Rahmat. DDLR Face Recognition use the help of face_recognition library to recognize faces, opencv to capture frame and flask for web application. DDLR Face Recognition was made for our final project data science introduction class.

You can try it by yourself, just make sure you have one of our team's face
